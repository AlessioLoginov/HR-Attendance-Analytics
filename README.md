# HR Attendance Analytics — Аналитика посещаемости сотрудников (FEML)

## Описание проекта
Проект посвящен исследованию данных о сотрудниках предприятия и анализу их посещаемости.  
Цель — применить методы машинного обучения для выявления закономерностей, влияющих на опоздания и отсутствие на рабочем месте, а также построить интерпретируемую модель, которая поможет HR-подразделению принимать более обоснованные кадровые решения.

Работа выполняется в рамках темы **FEML — “Работа с признаками и построение моделей”**.

---

## Цели проекта
- провести исследование датасета;
- изучить структуру и динамику посещаемости;
- выявить признаки, влияющие на опоздания и пропуски;
- построить ML-модель для прогнозирования отсутствия;
- подготовить аналитические выводы для HR-подразделения.

---

## Датасеты проекта

### 1. employees.csv
Содержит базовые характеристики сотрудников:
- ID сотрудника  
- возраст  
- должность  
- подразделение  
- стаж  
- дата найма  
- удаленность от места работы
- тип занятости  

### 2. attendance.csv
Фактическая информация о посещаемости:
- employee_id  
- дата  
- присутствие (1/0)  
- факт опоздания  
- время прихода / ухода  


## Используемые методы (FEML)

### Предобработка данных
- очистка и проверка данных  
- объединение таблиц  
- обработка пропусков  
- преобразование типов  
- генерация признаков:  
  - частота опозданий  
  - среднее отклонение от графика  
  - процент пропусков  
  - регулярность посещений   

### Моделирование
Используемые модели:
- Logistic Regression  
- Random Forest  
- Gradient Boosting (CatBoost, XGBoost)  
- Decision Tree  
- Другие модели

### Метрики
- accuracy  
- precision  
- recall  
- F1-score  
- ROC-AUC

1. Сбор и проверка датасетов  
2. Разведочный анализ данных (EDA)

4. Создание и обучение моделей

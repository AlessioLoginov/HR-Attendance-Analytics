# HR Attendance Analytics (FEML)

## Описание проекта
Проект посвящен анализу посещаемости сотрудников и построению ML-модели для прогнозирования **наличия хотя бы одного дня отсутствия в горизонте 3 месяцев** на основе исторических данных.

Работа выполнена в рамках темы **FEML — работа с признаками и моделями**, с фокусом на feature engineering и сравнении алгоритмов классификации.

Ноутбук проекта:  
**`Employee_Absence_Prediction_Final.ipynb`**

---

## Состав проекта и выполненные этапы

### 1. Подготовка данных
- загружены датасеты `employees.csv` и `attendance.csv`
- проверена корректность данных, удалены дубликаты
- приведены типы данных, обработаны пропуски
- таблицы объединены по `employee_id`

### 2. Разведочный анализ (EDA)
- проанализирована структура посещаемости
- выявлена сильная несбалансированность классов
- подтверждена информативность поведенческих признаков

### 3. Генерация признаков (FEML)
На основе истории посещаемости сформирован набор из **27 признаков**, включая:
- общий процент отсутствий
- долю отсутствий по дням недели
- частоту опозданий
- показатели регулярности посещений

Признаки рассчитаны **только по историческому периоду**, без утечки таргета.

### 4. Моделирование
Решалась задача бинарной классификации.

Использованные модели:
- Logistic Regression (baseline)
- Decision Tree
- Random Forest
- XGBoost
- XGBoost с подбором гиперпараметров
- Ensemble (Random Forest + XGBoost)

### 5. Оценка качества
Метрики:
- accuracy
- precision / recall
- F1-score
- ROC-AUC (ключевая)

---

## Результаты моделей (ROC-AUC, test)

| Модель                     | ROC-AUC |
|----------------------------|---------|
| Logistic Regression        | 0.656   |
| Decision Tree              | 0.621   |
| Random Forest              | 0.701   |
| XGBoost                    | 0.685   |
| XGBoost (tuned)            | 0.708   |
| Ensemble (RF + XGB)        | **0.733** |

Лучший результат показала ансамблевая модель.

---

## Вывод
Проект демонстрирует, что **история посещаемости** является информативным источником признаков для прогнозирования будущих отсутствий сотрудников. Использование ансамблей позволяет повысить качество по сравнению с отдельными моделями.

Результаты могут применяться для аналитической поддержки HR-подразделений при планировании ресурсов и профилактике рисков отсутствий.
